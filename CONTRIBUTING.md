# Contributing

Thanks for your interest in contributing to Aura Glow.

## Getting started

- Fork the repository
- Create a feature branch
- Keep changes focused and minimal
- Open a pull request with a clear description of what changed

## Development

This project is a static HTML experience.

- Main entry: `index.html`
- Preview asset: `preview.png`
- GitHub Pages workflow: `.github/workflows/pages.yml`

To test locally, open `index.html` in your browser.

## Pull request guidelines

- Preserve the visual style and smoothness of the UI
- Avoid layout shifts
- Keep the project easy to preview as a static site
- Update the `README.md` if behavior or setup changes
